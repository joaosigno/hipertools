Propriedade Size

Descrição

 

Especifica o tamanho da fonte com a qual o texto deve ser exibido.

Sintaxe

 

objeto. Size [= Integer]

O argumento objeto sempre será o objeto Font.

Comentários

 

O código abaixo ilustra o uso da propriedade Size.

Exemplo

 

Font.Size = 3
