Método SepDateOnClient

Descrição

 

Método que implementa no cliente, o Browser, um código em JavaScript que valida uma data separada em três campos.

Sintaxe

 

objeto.SepDateOnClient String1, String2, String3, String4, String5, [Boolean], [String6]

Parte Descrição
Objeto Requerido. Sempre terá que ser um objeto Validate.
String1 a String3 Requerido. Nome de cada um dos três campos da data. Estes campos são definidos com os métodos do objeto Form.
String4 Requerido. Nome do formulário definido utilizando-se o método Form.BeginForm
String5 Requerido. Nome da função a ser gerada.
Boolean Opcional. Define se a primeira data deve ser maior do que hoje, isto é, depois desta data. O padrão é falso (False).
String6 Opcional. Mensagem de erro apresentada caso a data seja menor do que hoje.

Comentários

 

O código abaixo ilustra o uso do método SepDateOnClient.

Exemplo

 

Form.BeginForm "meuAplicativo.asp?O=XY", "frmFormulario", , , "validaDates()"

Show.HTML "Data de Início:"

Form.Text "txtDia1", 2

Form.Text "txtMes1", 2

Form.Text "txtAno1", 4

 

 

Form.Submit "btnSubmit", "Enviar"

Validate.SepDateOnClient "txtDia1", "txtMes1", "txtAno1", "frmFormulario", "validaDates", True, "A Data de Início deve ser depois de hoje"

 

Form.EndForm
