Método EMail

Descrição

 

Valida se um E-mail é válido

Sintaxe

 

objeto.EMail String

Parte Descrição
Objeto Requerido. Sempre terá que ser um objeto Validate.
String Requerido. O email a ser validado.

Comentários

 

O código abaixo ilustra o uso do método EMail.

Exemplo

 

Dim strEMail

strEMail = "hipertools@hipertools.com.br"

 

If Not Validate.EMail(strEMail) Then

  Response.Write "Email inválido"

End If
