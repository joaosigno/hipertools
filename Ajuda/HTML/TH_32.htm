Objeto Character

 

O objeto Character é utiilzado para a exibição de texto utilizando imagens para desenho dos caracteres. Esse objeto é particularmente útil para gerar contadores de acesso.

Sintaxe

 

Set Character = Server.CreateObject("Hipertools30.Character")
