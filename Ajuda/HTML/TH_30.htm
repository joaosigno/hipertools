Instalação

A instalação é muito simples, sem precisar de grandes ajustes no seu computador. Basta colocar o CD-ROM e executar o SETUP.EXE.

O SETUP.EXE fara boa parte do processo de instalação será executado pelo programa de instalação e você precisará somente configurar os diretórios onde seus sites vão funcionar no IIS/PWS.

Necessidades mínimas recomendadas

 

Hardware
Processador: Pentium 133
Memória: 64MB
HD: 2GB

Software
SO Windows NT 4.0 SP4 ou Windows 95/98
HTTP Server IIS 4.0 ou PWS
Componente BD MDAC 2.1
MS-Access
MS-SQL 6.5 SP5

Estrutura de diretórios recomendada:

 

Crie um diretório no seu servidor HTTP (Web), com o nome HiperTools® de acordo com a sua configuração. Nossa configuração tem a seguinte estrutura de diretórios:

Diretório Descrição
D:\Inetpub Diretório raiz do IIS 4.0 ou PWS
D:\Inetpub\wwwroot Diretório raiz do servidor Web
D:\Inetpub\wwwroot\components Diretório para todos os componentes do servidor
D:\Inetpub\wwwroot\HiperTools Diretório para todos os arquivos do HiperTools® no servidor
D:\Inetpub\wwwroot\Clients Diretório para os sites dos clientes
D:\Inetpub\wwwroot\Projects Diretório para os sites dos projetos internos

Como padrão, o HiperTools® deverá instalar os arquivos Include no diretório abaixo ou equivalente, de acordo com a sua instalação do IIS/PWS.

Atualizando o HiperTools® v3.0:

 

Para atualizar o HiperTools® você terá 2 formas:

1 – Utilizando o programa de atualização, muito similar com o de instalação;

2 – Fazendo o registro do componente manualmente.

Esta parte explica como fazer a atualização do HiperTools® manualmente.

No prompt do DOS, vá no diretório do HiperTools® e procure o arquivos HIPERTOOLS30.DLL e digite o comando abaixo, que o registro do HiperTools® será cancelado. Para executar esta tarefa, o seu servidor IIS/PWS precisará estar completamente parado.

REGSVR32 /u HIPERTOOLS30.DLL

Depois, copie sobre o arquivo do HiperTools® a .DLL atualizada, no prompt do DOS digite o comando abaixo, que o HiperTools® será registrado novamente e estará pronto para o uso.

REGSVR32 HIPERTOOLS30.DLL

Obs: Caso você não encontre o programa regsvr32.exe no seu disco rígido, procure no nosso servidor de http://www.hipertools.com.br/ftp

Configurando um site no servidor IIS:

 

Recomendamos sempre utilizada um domínio virtual para cada projeto que você for criar dentro do seu servidor. Desta forma, você poderá controlar todos os ambientes, além de ter um global.asa diferente para cada projeto.
