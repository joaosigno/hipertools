Método BeginJS

Descrição

 

Inicia o sistema de validação no cliente usando JavaScript. Ele verifica se já foi iniciado ou aberto a TAG <SCRIPT>, se não estiver aberta – propriedade JSStarted – então abre faz com que JSStarted seja True

Sintaxe

 

objeto.BeginJS

Parte Descrição
Objeto Requerido. Sempre terá que ser um objeto Validate.

Comentários

 

O código abaixo ilustra o uso do método BeginJS.

Exemplo

 

If Not Validate.JSStarted Then

  Validate.BeginJS

End If

 

Response.Write "function fazAlgo(){"

Response.Write "  if(true){"

Response.Write "    ..."

Response.Write "  }"

Response.Write "}"

 

Validate.EndJS 'Aqui JSStarted volta a ser False

 

Aplica-se

 

Validate
