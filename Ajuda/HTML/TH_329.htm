Propriedade Item

Descrição

 

Retorna o objeto RegistryValue associado ao item especificado. Essa propriedade é read-only.

 

Sintaxe

 

objeto.Item(Variant)

 

Parte Descrição
objeto Será sempre uma referência a um objeto RegistryValues.
Variant String representando o nome do valor ou Integer indicando o índice.

 

Comentários

 

O código abaixo ilustra o uso da propriedade Item.

 

Exemplo

 

With Registry

If .Values(intIdx).Name = .Values(“acesso”).Name Then

Show.HTMLCR “O índice de acesso é ” & intIdx

End If

End With
