Método CompareDate

Descrição

 

Compara duas datas e retorna valores definidos pelas constantes de retorno.

Sintaxe

 

objeto.CompareDate String1, String2

Parte Descrição
Objeto Requerido. Sempre terá que ser um objeto Validate.
String1 Requerido. Primeira data representada por string ou por Date.
String2 Requerido. Segunda data representada por string ou para Date.

Comentários

 

Valores de retorna para este método:

Constante Valor Descrição
vlCmpDateFirstInvalid 1 A primeira data é inválida.
vlCmpDateSecondInvalid 2 A segunda data é inválida.
vlCmpDateBothInvalid 3 As duas datas são inválidas
vlCmpDateOutOfRange 4 As primeira data é maior de que a segunda data.

O código abaixo ilustra o uso do método CompareDate.

Exemplo

 

Dim dtmData1, dtmData2, intValid

 

dtmData1 = "20/11/1999"

dtmData2 = "19/11/1999"

 

intValid = Validate.CompareDate(dtmData1, dtmData2)

 

Response.Write intValid

'Retorna: 4

 

dtmData1 = "Validate"

dtmData2 = "19/11/1999"

 

intValid = Validate.CompareDate(dtmData1, dtmData2)

 

Response.Write intValid

Retorna: 1

 

dtmData1 = "20/11/1999"

dtmData2 = "CompareDate"

 

intValid = Validate.CompareDate(dtmData1, dtmData2)

 

Response.Write intValid

Retorna: 2
