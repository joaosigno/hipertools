Método WriteLink

Descrição

 

Inclui no documento as tags LINK.

 

Sintaxe

 

objeto.WriteLink

Parte Descrição
objeto Requerido. Sempre terá que ser um objeto Default.

Comentários

 

O código abaixo ilustra o uso do método WriteLink.

Exemplo

 

Default.BeginHTML

Default.BeginHead "Suporte de Informática"

Default.WriteMeta

Default.EndHead

Default.WriteBaseFont

Default.WriteLink

 

Default.IgnoreBodyWidth = True

Default.CenterBody = True

Default.BeginBody

 

Show.Center

 

Table.BeginTable "65%"

Table.Style.Color2 = ""

Table.BeginRow 4

 

Table.ColumnVAlign = "Middle"

Table.Column "<img src=""/img/Zevallos.gif"">"

Table.Column "<Font Color=""DarkBlue"">Suporte de Informática"

 

Table.EndRow

Table.EndTable

 

Default.PageFooterDefault

Default.EndBody

 

Default.EndHTML
