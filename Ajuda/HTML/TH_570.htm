Método CompareFieldsOnClient

Descrição

 

Método que implementa no cliente, o Browser, um código em JavaScript que valida a comparação de dois campos. A comparação é somente uma verificação de diferença entre os campos, se forem diferentes então retorna a mensagem de error adequada.

Sintaxe

 

objeto.CompareFieldsOnClient String1, String2, String3

Parte Descrição
Objeto Requerido. Sempre terá que ser um objeto Validate.
String1 Requerido. Nome do primeiro campo a ser comparado definido com algum método do objeto Form – Form.FormInput, Form.Text, etc.
String2 Requerido. Nome do segundo campo a ser comparado
String3 Requerido. String3 é a mensagem que será apresentada quando os campos forem diferentes.

Comentários

 

O código abaixo ilustra o uso do método CompareFieldsOnClient.

Exemplo

 

Form.BeginForm "meuApp.asp?O=XY", "frmFields", , , "vldFields()"

Show.HTML "Senha:"

Form.Text "txtSenha", 8, ,8 'Maxlenght 8

 

Form.Hidden "hidSenha", "1234567"

 

Form.Submit "btnSubmit", "Enviar"

Validate.CompareFieldsOnClient "txtSenha", "hidSenha", "frmFields", "vldFields", "Senha inválida"

 

Form.EndForm
