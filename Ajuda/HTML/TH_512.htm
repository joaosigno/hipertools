Propriedade IsLevelID

Descrição

 

Retorna verdadeiro se este nível for o nível clicado.

Sintaxe

 

[Result =] objeto.IsLevelID

O argumento objeto sempre será o objeto TreeView. O valor para Result é o IsLevelID, o valor de IsLevelID indica se este é o nível clicado.
