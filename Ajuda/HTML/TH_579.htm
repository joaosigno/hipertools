Método IsDate

Descrição

 

Valida se uma data, sem a hora incluída, é valida.

Sintaxe

 

objeto.IsDate String

Parte Descrição
Objeto Requerido. Sempre terá que ser um objeto Validate.
String Requerido. Representante Data a ser validada.

Comentários

 

O código abaixo ilustra o uso do método IsDate.

Exemplo

 

Dim strDate

 

strDate = "Juca"

 

If Not Validate.IsDate(strDate) Then

  Response.Write "Data inválida!!"

  'Entrará com certeza

End If

 

strDate = "15/06/1965"

 

If Not Validate.IsDate(strDate) Then

  Response.Write "Data inválida!!"

End If
