Propriedade TableBorder

Descrição

 

Define ou retorna se as tabelas apresentadas contém bordas.

 

Sintaxe

 

objeto.TableBorder [= String]

O argumento objeto sempre terá que ser um objeto Default.

O argumento String especifica a espessura da borda da tabela.

 

Comentários

 

O objeto Default altera o valor desta propriedade quando se altera o parâmetro de QueryString “border”.

O código abaixo ilustra o uso da propriedade TableBorder.

Exemplo

 

Default.TableBorder = "2"
