Método Center

Descrição

 

Marca o início de uma seção na qual os elementos do documento são centralizados.

Sintaxe

 

objeto.Center

Parte Descrição
Objeto Requerido. Sempre terá que ser um objeto Show.

Comentários

 

O código abaixo ilustra o uso do método Center.

Exemplo

 

Show.Center

Retorna: <CENTER>

 

Aplica-se

 

Show
