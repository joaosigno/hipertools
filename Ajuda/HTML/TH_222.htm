Propriedade Bold

Descrição

 

Especifica se o texto deve ser exibido em negrito.

Sintaxe

 

objeto. Bold [= Boolean]

O argumento objeto sempre será o objeto Font.

Comentários

 

O código abaixo ilustra o uso da propriedade Bold.

Exemplo

 

Font.Bold = True
