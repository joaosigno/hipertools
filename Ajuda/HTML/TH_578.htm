Método EndJS

Descrição

 

Termina o sistema de validação no cliente usando JavaScript. Neste método se altera o estado da Propriedade JSStarted

Sintaxe

 

objeto.EndJS

Parte Descrição
Objeto Requerido. Sempre terá que ser um objeto Validate.

Comentários

 

O código abaixo ilustra o uso do método EndJS.

Exemplo

 

Dim Validate

Set Validate = Server.CreateObject("HiperTools30.Validate")

If Not Validate.JSStarted Then

  Validate.BeginJS

End If

 

Response.Write "function fazAlgo(){"

Response.Write "  if(true){"

Response.Write "    ..."

Response.Write "  }"

Response.Write "}"

 

Validate.EndJS ‘Aqui JSStarted volta a ser False

 

Aplica-se

 

Validate
