Propriedade SetLevelID

Descrição

 

Define um valor de identificação deste para o próximo item apenas. É utilizada para posterior identificação por parte do usuário do item clicado.

Sintaxe

 

objeto.SetLevelID [= String]

O argumento objeto sempre será o objeto TreeView. O valor para SetLevelID é o identificador deste nível.
