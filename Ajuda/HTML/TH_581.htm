Método IsFieldEmptyOnClient

Descrição

 

Método que implementa no cliente, o Browser, um código em JavaScript que verifica se um campo está vazio – não preenchido.

Sintaxe

 

objeto.IsFieldEmptyOnClient [String]

Parte Descrição
Objeto Requerido. Sempre terá que ser um objeto Validate.
String Opcional. O Nome da funçào que verifica se o campo está vazio.

Comentários

 

O código abaixo ilustra o uso do método IsFieldEmptyOnClient.

Exemplo

 

Form.BeginForm "meuApp.asp?O=XY", "frmFields", , , "vldAll()"

Show.HTML "Senha:"

Form.Text "txtSenha", 8, ,8 'Maxlenght 8

 

Form.Hidden "hidSenha", "1234567"

 

Form.Submit "btnSubmit", "Enviar"

 

Validate.IsFieldEmptyOnClient "vldVazio"

 

Validate.BeginJava "vldAll", "frmFields"

  Show.HTML "if(!vldVazio()){"

  Validate.ClearFieldOnClient "txtSenha"

  Show.HTML "else{"

Validate.EndJava
