Método CPF

Descrição

 

Verifica se um CPF é válido

Sintaxe

 

objeto.CPF String

Parte Descrição
Objeto Requerido. Sempre terá que ser um objeto Validate.
String Requerido. Representação em string do CPF. Pode vir com caracteres especiais ou não.

Comentários

 

O código abaixo ilustra o uso do método CPF.

Exemplo

 

Dim strCPF

 

strCPF = "1.111.111.111-1"

 

If Not Validate.CPF(strCPF) Then

  Response.Write "CPF inválido!!"

End If
