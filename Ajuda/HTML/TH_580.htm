Método IsDateOnClient

Descrição

 

Método que implementa no cliente, o Browser, um código em JavaScript que valida uma data.

Sintaxe

 

objeto.IsDateOnClient String1, String2, String3, [Boolean], [String4]

Parte Descrição
Objeto Requerido. Sempre terá que ser um objeto Validate.
String1 Requerido. Nome do campos que contém a data. Definidos com os métodos do objeto Form.
String2 Requerido. Nome do formulário definido utilizando-se o método Form.BeginForm
String3 Requerido. Nome da função a ser gerada.
Boolean Opcional. Define se a primeira data deve ser maior do que hoje, isto é, depois desta data. O padrão é falso (False).
String4 Opcional. Mensagem de erro apresentada caso a data seja menor do que hoje.

Comentários

 

O código abaixo ilustra o uso do método IsDateOnClient.

Exemplo

 

Form.BeginForm "meuAplicativo.asp?O=XY", "frmDate", , , "validaDates()"

Show.HTML "Data de Início:"

Form.Text "txtData", 12

 

 

Form.Submit "btnSubmit", "Enviar"

Validate.IsDateOnClient "txtData", "frmDate", "validaDates", True, "A Data deve ser depois de hoje"

 

Form.EndForm
