Método IsRequiredOnClient

Descrição

 

Método que implementa no cliente, o Browser, um código em JavaScript que verifica se um campo está vazio, e se ele já estiver vazio faz com que a função corrente retorne falso.

Este método depende da chamada do método IsFieldEmptyOnClient.

Sintaxe

 

objeto.IsRequiredOnClient String1, String2

Parte Descrição
Objeto Requerido. Sempre terá que ser um objeto Validate.
String1 Requerido. Nome do campo definido com algum método do objeto Form – Form.FormInput, Form.Form.Text, etc.
String2 Requerido. Mensagem que será apresentada no caso de erro.

Comentários

 

O código abaixo ilustra o uso do método IsRequiredOnClient.

Exemplo

 

Form.BeginForm "meuApp.asp?O=XY", "frmFields", , , "vldAll()"

Show.HTML "Senha:"

Form.Text "txtSenha", 8, ,8 'Maxlenght 8

 

Form.Hidden "hidSenha", "1234567"

 

Form.Submit "btnSubmit", "Enviar"

 

Validate.IsFieldEmptyOnClient "vldVazio"

 

Validate.BeginJava "vldAll", "frmFields"

  Validate.IsRequiredOnClient

  Show.HTML "if(!vldFields()){"

  Validate.ClearFieldOnClient "txtSenha"

  Show.HTML "  return false;"

  Show.HTML "}"

Validate.EndJava

 

Validate.CompareFieldsOnClient "txtSenha", "hidSenha", "frmFields", "vldFields", "Senha inválida"

 

Form.EndForm
