Propriedade SourceCode

Descrição

 

Define ou retorna, se o valor não for vazio, se apresenta-se o ícone que aponta para o apresentador de código-fonte. Este apresentador.se encontra no diretório virtual do HiperTools.

 

Sintaxe

 

objeto.SourceCode [= Boolean]

O argumento objeto sempre terá que ser um objeto Default.

 

Comentários

 

O código abaixo ilustra o uso da propriedade SourceCode.

Exemplo

 

Default.SourceCode = True
