Objeto TableStyle

 

Objeto que implementa um conjunto de propriedades utilizadas na formatação de tabelas geradas com o objeto Table, e também pelo objeto EditForm.

Sintaxe

 

Set TableStyle = Server.CreateObject("HiperTools30.TableStyle")

Set Table.Style = TableStyle

Set Edit.Style = TableStyle
