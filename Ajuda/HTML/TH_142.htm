Propriedade Errors

Descrição

 

Retorna uma referência para uma coleção do tipo “ADODB.Errors” com todos os erros que ocorreram no momento. Veja a documentação padrão do ADO.

 

Sintaxe

 

objeto.Errors

 

O argumento objeto sempre será o objeto EditForm.

 

Comentários

 

O código abaixo ilustra o uso da propriedade Errors.

 

Exemplo

 

Dim objErrors, objError

Set objErrors = Server.CreateObject("ADODB.Errors")

 

Edit.DebugMode = True

 

' Definição dos dados...

.

.

.

 

If Edit.Errors.Count > 0 Then

Set objErrors = Edit.Errors

For Each objError In objErrors

Response.Write objError.Number & " " & objError.Description

Next

 

End If
