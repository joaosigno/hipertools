Método IsTime

Descrição

 

Verifica se uma hora é válida.

Sintaxe

 

objeto.IsTime String

Parte Descrição
Objeto Requerido. Sempre terá que ser um objeto Validate.
String Requerido. Hora a ser validada

Comentários

 

O código abaixo ilustra o uso do método IsTime.

Exemplo

 

Dim strTime

 

strTime = "Juca"

 

If Not Validate.IsTime(strDate) Then

  Response.Write "Horário inválido!!"

  'Entrará com certeza

End If

 

strDate = "12:35"

 

If Not Validate.IsTime(strTime) Then

  Response.Write "Horário inválido!!"

End If
